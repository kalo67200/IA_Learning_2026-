# Knowledge Base Schema

## What This Is
Ma base de connaissances personnelle sur l'IA et le Machine Learning.

## How It's Organized
- raw/ : sources brutes — articles, notes, explications. Ne jamais modifier.
- wiki/ : wiki organisé maintenu par l'IA.
- outputs/ : réponses, résumés, analyses générés.

## Wiki Rules
- Chaque concept a son propre fichier .md dans wiki/
- Chaque fichier commence par un résumé en une phrase
- Maintenir un INDEX.md avec tous les sujets
- Liens entre sujets avec [[nom-du-sujet]]

- Machine Learning et Deep Learning
- Réseaux de neurones et backpropagation
